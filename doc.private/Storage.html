Module Storage

module Storage: sig .. end

This file includes the various scenarios and choices that the player can make (and their consequences).

None of this should be exposed to the client!! It is exposed to implementers because it is used in other files. This is why it does not have an mli, and does not show up on makedocs

val decision_consequence_list : (string * (string * float) list) list

decision_consequence_list is a list of the various choices and then the various corresponding consequences

val decision_scenario_name : (string * string) list

decision_scenario_name is a list of choices and then the corresponding next scenario

val scenario_friends_list : (string * string) list

scenario_friends_list is a list of friends that can be gained at certain circumstances

val has_prereq : string list

has_prereq is a list of scenarios that need to be unlocked previously

val unlock_list : (string * (string * string)) list

unlock_list is a list of scenarios that need to be unlocked previously, and their special consequences

val friend_closeness_list : (string * (string * int)) list

friend_closeness_list is a list of the relative closeness of friends at different choices
