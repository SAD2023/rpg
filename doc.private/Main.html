The main entry point for the Big Red Redemption game

val give_number : string -> int

give_number letter returns a number that corresponds to a given letter. For instance, "A" corresponds to 0, "B" corresponds to 1 etc...

play_game f keeps the game playing.

val play_minigames : Student.student -> Scenario.scenario -> int -> unit

play_minigames player scenario acc will have the player play a minigame every ten scenarios

val your_qualities : unit -> unit

your_qualities () prints a description of the different qualities that the player will be jedged for onto the GUI

val main : unit -> unit

main () gives the player a description of the game, including the qualities that they will be judged for. It then asks them for their name and then starts the engine to run the rest of the program.
