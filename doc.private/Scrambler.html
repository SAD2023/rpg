Module Scrambler

module Scrambler: sig .. end

Scrambler is a very simple minigame that prints out a word arranged in a random order, and then asks the player to unscramble it

val words_to_scramble : string list

words_to_scramble is just a list of words to scramble for the minigame

val word_picker : 'a list -> 'a

word_picker lst_of_words picks a random word from an given list of words

val remove_char : string -> int -> string

remove_char str index takes in a string and index value and removes a character in order to switch up the ordering of a word.

val scramble_word : string -> string -> string

scramble_word takes in a string word and scrambles the given word with the remove_char helper function.

val scramble_intro : string -> unit

scramble_intro prints the introduction/instructions for the scramble minigame.

val scramble_engine : string -> string -> string

scramble_engine takes in a word and a user's input and checks to see if the user's input correctly matches the word.

val play_minigame : Student.student -> int -> Student.student * int

play_minigame student acc plays the scrambler minigame
