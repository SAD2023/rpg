We will assume you already have OCaml and opam installed.

You do not need to install any other external libraries at this moment, besides
the OCaml graphics module using opam. 

You might also need to install XMing (Windows) or XQuartz (Macs).

Simply download the code, and run make build to compile.

To play the game:
  - Run make play from the terminal.


If you have any trouble installing it, please contact any of the authors and we 
will reach out to offer you assistance!
